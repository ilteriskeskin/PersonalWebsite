Python ile Emoji Modülü

Posted on Cum 16 Ağustos 2019 in Python3

Merhaba arkadaşlar, bu yazımda eğlenceli bir modül olduğunu düşündüğüm emoji modülünden bahsedeceğim. Python ile yazdığınız konsol veya web uygulamalarında emoji kullanmanıza olanak sağlayan tatlı bir proje.

Emoji Modülü Kurulumu

pip ile her zaman olduğu gibi modülümüzü kuruyoruz.

pip3 install emoji

Kullanım

Peki ya şimdi? E madem kurduk küçük bir konsol uygulamasıyla bunu test edelim. konsol.py isimli bir dosya açalım ve içine aşağıdakileri yazalım.

import emoji

print(emoji.emojize('Python is :thumbs_up:'))

uygulamanızı çalıştırdığınızda konsol ekranınızda bir onay emojisi göreceksiniz..

Peki bir Flask uygulamasında nasıl oluyor o işler? Hemen ufak bir örnekle bakalım nasıl oluyor. app.py isimli bir dosya açalım ve içine aşağıdakileri yazalım.

from flask import Flask
import emoji

app = Flask(__name__)


@app.route('/')
def home():
    return emoji.emojize('Water! :grinning_face:')


if __name__ == '__main__':
    app.run(debug=True)

uygulamanızı çalıştırdığınızda ve 127.0.0.1:5000 adresine girdiğinizde gülen bir surat sizi karşılayacak.

Emoji listelerine burdan ulaşabilirsiniz. Orada yazan isimleri _ ile birleştirip kullanabilirsiniz.

Bu yazımızı burada sonlandıralım arkadaşlar. Umarım faydalı olmuştur.

Herkese iyi çalışmalar diliyorum :)
